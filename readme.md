**# Iris Classification Project**



**## Overview:**

This project predicts iris species using sepal and petal features.



**## Steps:**

\- Load dataset (`data/iris.csv`)

\- Perform EDA and visualizations

\- Train models (KNN, Logistic Regression, Decision Tree)

\- Save best model as `iris\_model.pkl`

\- Generate plots into `screenshots/`

\- Document requirements and summary



**## How to Run:**

1\. Install dependencies:

2\. Open `IrisClassification.ipynb` in Jupyter Notebook.

3\. Run all cells.

4\. Saved model will appear as `iris\_model.pkl`.



**## Deliverables:**

\- Notebook

\- Model file

\- Screenshots

\- Executive summary
